# Codes for generating results and figures

* `01_prepare_power.ipynb` contains the code for generating monthly energy data from daily and hourly data available at https://github.com/dougrichardson/power_models and https://github.com/dougrichardson/demand_model. The former repository also contains the code for generating Fig. 1.
* `02_climatology.ipynb` contains the code for generating Figs. 2, 3 and 5.
* `03_power_correlations.ipynb` contains the code for generating Figs 4, 6 and 7.
